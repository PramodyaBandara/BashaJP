පාඩම 11: て Form (N4 ආරම්භය)

පාඩමේ අරමුණ

  • て form තේරුම් ගැනීම
  • actions connect කිරීම
  • requests කරන්න හැකි වීම

て Form කියන්නේ මොකක්ද?

て form කියන්නේ verb එක වෙනස් කරලා actions connect කරන form එකක්.

Basic Examples

  • たべます → たべて
  • のみます → のんで
  • いきます → いって

Use 1: Request (〜てください)

👉 polite request

  • たべてください → කන්න
  • まってください → ඉන්න

Use 2: Action Connection

👉 actions link කරනවා

  • たべて ねます → කෑවට පස්සේ නිදාගන්නවා
  • おきて いきます → නැගිටලා යනවා

Speaking Patterns

  • ___てください
  • ___て ___ます

Mini Dialogue

A: ちょっと まってください
B: はい

Practice

  • verbs 5ක් て form කරන්න
  • request sentences 3ක් කරන්න
  • daily actions connect කරන්න

Real-Life Tip

Japan වල polite requests → てください use කරන්න.
