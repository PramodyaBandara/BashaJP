පාඩම 07: A は B です / か

පාඩමේ අරමුණ

  • “A යනු B වේ” කියන වාක්‍ය pattern එක භාවිත කිරීම
  • ප්‍රශ්න වාක්‍ය සෑදීම
  • conversation flow එකක් ගොඩනගා ගැනීම

මූලික Pattern

A は B です
👉 A = topic
👉 B = description

උදාහරණ

  • わたしは がくせいです → මම ශිෂ්‍යයෙක්
  • わたしは にほんじんです → මම ජපන් කෙනෙක්

ප්‍රශ්න (Questions)

👉 වාක්‍ය අන්තිමට か දාන්න

  • あなたは がくせいですか？
  • あなたは せんせいですか？

Answers

  • はい、そうです → ඔව්
  • いいえ、ちがいます → නැහැ

Speaking Practice

  • わたしは ___ です
  • あなたは ___ ですか？
  • はい / いいえ practice කරන්න

Mini Dialogue

A: あなたは がくせいですか？
B: はい、そうです

Common Mistakes

  • か නැතුව question හදන්න එපා ❌
  • です skip කරන්න එපා ❌
