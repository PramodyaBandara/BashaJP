GitHub Pages publish tips

Repository එකට මෙම folders upload කර, Pages source එක root branch වෙත යොදා publish කරන්න.

Step 1

ZIP unzip කර repository එකකට upload කරන්න.

Step 2

GitHub Settings → Pages වෙත ගොස් source එක main branch / root ලෙස තෝරන්න.

Step 3

කිහිප මිනිත්තු තුළ public URL එකක් මත site එක live වේ.
